Welcome to Pimp! My App

I wrote this simple application for a set of presentations and events I did in fall 2010 and spring 2011 all around France, Switzerland and Luxembourg.

For my demo I use a local MySQL DB with 700k entries for BigTable and 50k for BigTableR. For the code source version hosted on GitHub, I configured a local db/fs with a small bootstrap script.
